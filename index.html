Research & Open Source
R Package · Forest ML CRAN Published v0.1.1
FuelDeep3D — 3D Forest Fuel Segmentation via TLS & Deep Learning

An R package published on CRAN that provides end-to-end workflows for processing, feature extraction, and deep learning-based classification of 3D forest point clouds from terrestrial laser scanning. Supports training-data creation, height-above-ground feature extraction, tiling and preprocessing of large TLS scenes, model inference on new LAS/LAZ files, and writing predicted fuel classes back to point clouds — enabling applications in forest structure assessment, wildfire behaviour modelling, and fuel complexity analysis. Integrates with the lidR ecosystem and uses a PyTorch/PointNext backend via reticulate.

RPythonPyTorchLiDAR / TLS PointNextCRANForest AnalyticsWildfire Modelling

rTLsDeep — Post-Hurricane Tree Damage Classifier

An R-based deep learning pipeline for classifying post-hurricane tree damage severity from terrestrial laser scanning data.
